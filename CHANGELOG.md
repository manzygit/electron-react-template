### Added
- Basic Electron + Vite + React + TypeScript setup
- Separated `main` and `renderer` TS configs
- Secure preload with typed API
- Shared types (`shared.ts`)
- Default IPC channels (`PING`, `GET_VERSION`)
- Working React UI example
- Live reload (Vite + Electron)
- Build system for renderer + main
- Electron-builder packaging config
- Template README
